Integration test for the aa_composition score term with the following features:
- global .comp file setup (with flags).
- local .comp file setup (with <Set> tags in RosettaScripts).
- Pose-level .comp file setup (with the AddCompositionConstraint and ClearCompositionConstraints movers).
- ResidueSelector-based control of amino acid composition (with the LayerSelector).

Author: Vikram K. Mulligan, Ph.D., Baker laboratory, University of Washington (vmullig@uw.edu).
